# CodeOrbit Tech Internship - Full Stack Development

## Project Title
CodeOrbit Landing Page

## Description
A responsive landing page built as part of the CodeOrbit Tech Full Stack Development Internship. This project showcases the implementation of clean UI design, responsive grids, and interactive web elements.

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Setup & Preview Instructions
1. Clone or download this repository to your local machine.
2. Open the `public/index.html` file in any modern web browser to view the landing page.
3. To view the local server setup (if applicable), open your terminal in the project directory, run `node server.js`, and visit `http://localhost:3000`.
